XTerm-manager

A good quick way to set up shell access methods on a workstation

Download this project as a .zip file Download this project as a tar.gz file

shell.sh

If you get a new workstation and to aid setting up terminal windows you'll need to juggle,

mkdir x cd x git clone https://github.com/topromulan/xterm-manager/

Then, edit the .txt files in x/shells/ that represent different shells for useful hostnames, and just connect a handy hot key to "x/shell.sh hostname" in your window-manager.
